Reset Wallet

MuLPay does not keep a copy of your password. If you are having problems unlocking your account and need to reset your wallet. You can do this by providing the mnemonic used to set up your wallet. This will remove your current wallet and mnemonic from the device, as well as the list of accounts you have compiled. After resetting using the mnemonic, you will see a list of accounts (based on the mnemonic used to reset). This new list will automatically include accounts that have a balance. You can also re-add any other accounts using the ones you created previously. The custom accounts you import will need to be re-added, as will any custom tokens added to the account.

Please ensure that you are using the correct tokens before proceeding. You will not be able to undo this action.
